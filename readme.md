## How to Run the Programs

To run the PHP scripts in this repository, you will need a local PHP development environment. We recommend using a stack like XAMPP, WAMP, or MAMP.

1. **Install a Local Server**: Download and install [XAMPP](https://www.apachefriends.org/) (or similar).
2. **Clone the Repository**: Clone this repository or download it as a ZIP file.
3. **Move to Server Directory**: Place the `bvimr-php` folder inside your server's root directory:
   * For XAMPP: `C:\xampp\htdocs\`
   * For WAMP: `C:\wamp\www\`
4. **Start the Server**: Open the XAMPP/WAMP Control Panel and start the **Apache** service (and **MySQL** if the script requires a database).
5. **Execute in Browser**: Open your web browser and navigate to:
   ```text
   http://localhost/bvimr-php/Lab%20File/q1.php
   ```
   *(Change `q1.php` to the name of the file you want to execute).*
